Sharing Unsteady Data in Web-Browsers

The advantage of visualising unsteady data in the browser is that the interactivity is retained: figures can be zoomed and panned on-the-fly. Furthermore, by visualising the data in the browser sharing the data while retaining interactivity is made easier: the recipient of the link to the visualisation doesn't need to have special software install, and the visualisations are drawn when the web-page loads.

The limitations are internet speed and browser memory. The following chart tells which unsteady flow-field datasets can be smoothly analysed in the browser and which not.
